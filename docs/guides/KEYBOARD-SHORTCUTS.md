# Keyboard Shortcuts

| Shortcut | Action |
|---|---|
| Ctrl+K / Ctrl+Space | Command Center |
| Esc | Close modal / palette |
| Enter | Next field (forms) |
| Ctrl+Enter / Alt+S | Save |
| Number keys | Quick module rail (where enabled) |

Desktop: standard Edit/View roles (Copy, Paste, Zoom, Fullscreen).
