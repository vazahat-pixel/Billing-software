# Quick Start Guide — Textile ERP v1.0

1. **Install** — Run `TextileERP-Setup-1.0.0.exe` or open the web app URL.
2. **Login** — Use owner credentials created at signup.
3. **Welcome Wizard** — Complete company profile, GST, CoA, masters (< 15 minutes).
4. **Activate License** — Online key or offline code (Utilities → Commercial Release).
5. **First bill** — Create a Party, Item, then Sales invoice (Ctrl+K to search).
6. **Backup** — Infra → Backups → Run Manual Backup.

Keyboard: **Ctrl+K** Command Center · **Ctrl+Enter** Save · **Enter** next field.
