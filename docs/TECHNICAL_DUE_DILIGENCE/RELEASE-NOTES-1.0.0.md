# RELEASE NOTES — 1.0.0 Commercial Launch

Textile ERP SaaS is certified for commercial distribution as **Version 1.0**.

## Highlights
- Full trading, inventory, job-work, accounting, GST stack (Stages 2–4)
- Enterprise productivity (Stage 6) & production infrastructure (Stage 7)
- Desktop Windows installer scaffold
- License activation (online/offline) & welcome onboarding
- Commercial readiness certification

## Operator checklist
1. Complete Welcome Wizard
2. Activate license
3. Run Commercial Certification (target ≥ 85)
4. Schedule backups
5. Distribute desktop installer or web URL
